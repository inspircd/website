Detailed Description

A structure that defines a command. Every command available in InspIRCd must be defined as derived from Command.

Create a new command.

Parameters:
InstancePointer to creator class
cmdCommand name. This must be UPPER CASE.
flagsUser mode required to execute the command. May ONLY be one mode - it's a string to give warnings if people mix params up. For oper only commands, set this to 'o', otherwise use 0.
minparaMinimum parameters required for the command.
maxparaMaximum number of parameters this command may have - extra parameters will be tossed into one last space-seperated param.
before_regIf this is set to true, the command will be allowed before the user is 'registered' (has sent USER, NICK, optionally PASS, and been resolved).
