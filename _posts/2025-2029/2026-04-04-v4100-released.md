InspIRCd v4.10.0 is now available, [download it from the releases page](https://github.com/inspircd/inspircd/releases/tag/v4.10.0).

The most notable changes in this release are:

- Added extended ban `d:` (redirect) to the redirect module.

- Added support for the IRCv3 ACCOUNTEXTBAN ISupport token.

- Added support for the IRCv3 no-implicit-names specification to the ircv3 module.

- Added support for the IRCv3 UTF8ONLY specification.

- Added support for validating extended bans before they are set.

- Added the sharebans module which allows bans to be shared between channels.

<!--more-->

No breaking configuration changes have been made since the v4.0.0 release. Please consult [the list of breaking changes](https://docs.inspircd.org/4/breaking-changes) if you are upgrading from v3.

The full change log can be found [on the docs site](https://docs.inspircd.org/4/change-log#inspircd-4100) or [on GitHub](https://github.com/inspircd/inspircd/compare/v4.9.0...v4.10.0).
