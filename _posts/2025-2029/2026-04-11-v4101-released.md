InspIRCd v4.10.1 is now available, [download it from the releases page](https://github.com/inspircd/inspircd/releases/tag/v4.10.1).

The most notable changes in this release are:

- Fixed the Windows binary containing the wrong version in the main executable.

- Relaxed validation of usernames in `USER` as it broke some clients.

- Whitelisted the `+channel-context` tag in the ircv3_ctctags module.

<!--more-->

No breaking configuration changes have been made since the v4.0.0 release. Please consult [the list of breaking changes](https://docs.inspircd.org/4/breaking-changes) if you are upgrading from v3.

The full change log can be found [on the docs site](https://docs.inspircd.org/4/change-log#inspircd-4100) or [on GitHub](https://github.com/inspircd/inspircd/compare/v4.10.0...v4.10.1).
